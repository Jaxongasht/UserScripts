# UserScripts / Пользовательские сценарии


Репозиторий скорее для личного пользования, хотя большинство скриптов (а может быть и все =))
под лицензией [WTFPL](http://www.wtfpl.net/), так что милости прошу.


## Что и где?

Скрипты лежат преимущественно в каталогах, посвящённых отдельным сайтам

## Почему GitHub?

* [Userscripts.org](http://userscripts.org:8080/) похоже собирается отойти.
* Приходящий ему на смену [Greasy fork](http://greasyfork.org/) пока что не вполне устаканился и
  в целях безопасности вытворяет
  [какие-то немного странные вещи](https://greasyfork.org/forum/discussion/comment/2464/#Comment_2464).
* В силу специфики жанра публиковать эти скрипты где-то надо =).

В итоге получается, что, кроме как на GitHub, больше негде =).
